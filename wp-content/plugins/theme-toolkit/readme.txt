== Description ==

Theme toolkit is a plugin to register custom post types, widgets and shortcodes to add additional feature and functionality to any WordPress theme. It supports testimonial, portfolio, team and partners custom post types.

Demo Available [Here](https://preciousthemes.com/demo/theme-toolkit) 


=  Team Shortcodes =
[tt-team]
[tt-team post_number="x"]
[tt-team column="x"] where x can be 2,3 or 4
[tt-team image_size="size"] where size can be thumbnail, medium, large, full or size defined in theme

= Portfolio Shortcodes =
[tt-portfolio]
[tt-portfolio post_number="x"]
[tt-portfolio column="x"] where x can be 2,3 or 4
[tt-portfolio image_size="size"] where size can be thumbnail, medium, large, full or size defined in theme

= Testimonials Shortcodes =
[tt-testimonials]
[tt-testimonials post_number="x"]

= Partners Shortcodes =
[tt-partners]
[tt-partners post_number="x"]
[tt-partners image_size="size"] where size can be thumbnail, medium, large, full or size defined in theme

== Installation ==

1. Go to Plugins > Add New and search for Theme Toolkit. 
2. Install and activate it.


== Frequently Asked Questions ==

= Does this plugin work for other themes ? =

Yes, it works for all themes.
